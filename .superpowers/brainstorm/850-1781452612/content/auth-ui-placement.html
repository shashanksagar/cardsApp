Where should the Login / Register UI live?

How does a user get to the login screen before starting their exam?

A

New screen before Start (Recommended)

A dedicated Login/Register screen appears first when the app loads. After signing in, the existing Start screen appears as normal. Clean separation.

Flow
Login/Register → Start Screen → Exam → Results
B

Modal overlay on the Start screen

The Start screen is always visible. A login modal pops up automatically if no user is logged in. After login it dismisses and the Start screen is usable.

Flow
Start Screen + Login Modal → Exam
C

Inline on the Start card (login fields embedded)

The login/register form is built directly into the Start card. When logged in, the form is replaced by a "Welcome, Name" section with a logout button.
