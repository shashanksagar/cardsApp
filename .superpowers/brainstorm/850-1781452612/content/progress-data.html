What progress data should be saved per user?

Every completed session is stored in the Node server's JSON file. Choose what gets recorded.

A

Session summary (Recommended baseline)

Date, question set, mode, score %, pass/fail, time taken. Enough to show a history table: "Set 2 · Exam · 74% · Passed · 14 Jun".

B

Topic breakdown per session

Per-topic scores saved alongside each session (e.g. "Identity Management 80%, OAuth 60%"). Lets you show topic trends over time.

C

Full question-level answers

Every question, what the user selected, and whether it was correct. Enables replaying any past session and seeing exactly which questions were missed.

You can select multiple options — they stack on top of each other.
