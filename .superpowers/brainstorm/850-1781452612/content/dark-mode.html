How should dark mode work?

Choose how users toggle between light and dark themes.

A

Toggle button in the UI (Recommended)

A 🌙/☀️ button visible on every screen (top-right corner). User preference saved to localStorage so it persists across sessions. Works regardless of OS setting.

Every screen — top right
B

Follow OS system preference automatically

Uses prefers-color-scheme media query. No toggle needed — app matches whatever the user's OS is set to. No user action required.

Pros

  • Zero UI clutter
  • Feels native

Cons

  • Can't override without changing OS
C

Both — OS default + manual override

App respects OS preference by default. A toggle button lets users override it per-session. Preference saved to localStorage. Most flexible but slightly more complex.

Pros

  • Best of both worlds

Cons

  • Slightly more logic
